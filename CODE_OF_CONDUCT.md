# Code of Conduct

## Our Standard

We expect all participants in this project to be respectful, constructive, and professional.

Examples of positive behavior include:

- Giving and accepting technical feedback with respect.
- Focusing criticism on ideas and implementation, not people.
- Being considerate of different backgrounds and communication styles.

Unacceptable behavior includes:

- Harassment, discrimination, threats, or personal attacks.
- Trolling, deliberate disruption, or bad-faith engagement.
- Publishing private information without permission.

## Scope

This Code of Conduct applies to:

- Repository issues, pull requests, and discussions.
- Project-related communication in public or private channels.

## Enforcement

Maintainers are responsible for clarifying and enforcing this policy.

For serious incidents, contact the maintainers privately via GitHub at
[@realmorrisliu](https://github.com/realmorrisliu).

Maintainers may remove comments, close discussions, or ban contributors for violations.
